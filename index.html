Beach Cleaning

In this project we will create a game in which we will collect as many plastic bags as we can to help clean the beach and win. Click in the example to play the game:

Click here

1) Create a new project, choose a backdrop and 2 objects, a character and an object you want to collect. If you you'd like you can use the same ones from the example.

2) When the game starts make the character go to a certain position and make him move to the left if the left arrow key is pressed and to the right if the right arrow key is pressed.

3) When you press the green flag make the bag or object you chose begin in a random position at the top of the screen and then make it fall continously.

Hint: Use the clone blocks to achieve this effect.

4) If the bag touches the bottom edge make it say "Thank you for cleaning the beach!" for 2 seconds and then make everything stop.

5) If the bag falls and touch your character make this clone disappear and create another clone that falls from above.

6) Create 2 variables, one for the level and the other one for the score. Set the level variable to start at 1 and the score variable to start at 0 when the green flag is pressed.

7) Add some code that makes the score change by 1 when the bag falls on your character.

8) When you press the green flag make it so when you reach a score of 3 the level goes up by 1 and the score returns to 0.

9) Add some code that as the level increases the bags fall faster.

Hint: There is always more than one way to obtain the same result. Good Luck!

Bonus: This project is a little more complex but make it to suit your own style, draw you own objects, make the backdrop change, add music or sounds. You can do it!

*** Important: Always remember to save your project and share it when you're done.
